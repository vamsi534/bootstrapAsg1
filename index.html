Day Wise Activites

Day 1. Basics of Internet

We shall discuss how the web works and how web applications operate. This will give a good context to entire course. Structure of web page : This will discuss HTML in detail. Web page in STYLE : This will discuss CSS in detail along with BootStrap the popular CSS Framework. mini hackathon to build a page and style it.

Day2. JavaScript :

Adding behavior to a page. DOM : manipulating page using code. jQuery: making your task of DOM manipulation a piece of cake. AJAX using jQuery and JSON and consuming a web service. Creating a small game using HTML, CSS and JavaScript. Day ends with another mini hackathon to have people innovate in whatever they want to make.

Day 3 : Django :

Django – introducing web server and starting a Django app. Databases – MySQL working with database in Django. Make Instagram clone in Django using whatever we learnt till now. Day ends with a full night hackathon asking people in teams to build an end to end app of their choice.
